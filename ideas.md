## Idea 1: "Quantitative Elegance" — Swiss Modernism meets Data Science

**Design Movement**: Swiss International Typographic Style fused with Quantitative Finance aesthetics

**Core Principles**:
1. Grid-based asymmetry — content blocks offset to create visual tension without chaos
2. Data as decoration — subtle financial chart motifs (candlestick, scatter, line) used as background texture
3. Restrained color with one sharp accent — ink-black, off-white, and a single electric teal
4. Typography as hierarchy — size and weight do all the work, no decorative flourishes

**Color Philosophy**:
- Background: #F5F4F0 (warm off-white, like aged paper)
- Text: #1A1A1A (near-black, authoritative)
- Accent: #00B4A6 (teal — signals data/tech without being flashy)
- Secondary: #8A8A8A (muted gray for metadata)
- Emotional intent: Serious, precise, trustworthy, quietly modern

**Layout Paradigm**:
- Left-anchored sidebar navigation (fixed, narrow) with name/title
- Main content area offset right, with generous margins
- Section headers use large typographic numbers (01, 02, 03) as visual anchors
- Horizontal rule dividers with subtle teal accent dots

**Signature Elements**:
1. Subtle SVG scatter plot pattern as hero background (low opacity)
2. Monospaced font for code/data snippets, contrasting with serif body
3. "Tag" chips for research keywords (ML, Asset Pricing, NLP)

**Interaction Philosophy**:
- Smooth scroll between sections
- Hover on research papers reveals abstract in a slide-out panel
- Navigation items highlight with a teal left-border on hover

**Animation**:
- Entrance: elements fade-in with slight upward translate (staggered, 0.1s delay each)
- Hover: subtle scale(1.01) on cards, no bounce
- Scroll-triggered: section numbers count up when entering viewport

**Typography System**:
- Display: "DM Serif Display" — authoritative, elegant for name/headings
- Body: "DM Sans" — clean, readable, modern
- Mono: "JetBrains Mono" — for code, equations, technical details
- Hierarchy: 48px display / 28px h2 / 18px h3 / 16px body / 13px caption

## Idea 2: "Dark Academia Finance" — Editorial Depth

**Design Movement**: Dark Academia meets Bloomberg Terminal aesthetics

**Core Principles**:
1. Dark, rich backgrounds with warm amber/gold accents — serious and atmospheric
2. Editorial layout — asymmetric columns, pull quotes, large section imagery
3. Subtle noise/grain texture on backgrounds for depth
4. Dense information hierarchy — like a well-designed academic journal

**Color Philosophy**:
- Background: #0F1117 (deep navy-black)
- Surface: #1A1D27 (slightly lighter for cards)
- Text: #E8E4D9 (warm cream, easy on eyes)
- Accent: #C9A84C (warm gold — finance, prestige)
- Secondary accent: #4A7FA5 (steel blue — data/ML)
- Emotional intent: Sophisticated, deep, intellectually serious

**Layout Paradigm**:
- Full-width hero with name large and centered, then asymmetric content below
- Two-column layout for research papers (title left, metadata right)
- CV as a timeline running down the page with year markers
- Hobbies section breaks the grid with a masonry photo layout

**Signature Elements**:
1. Grain/noise overlay on hero section (CSS filter)
2. Gold horizontal rule separators
3. Animated "typing" effect for the tagline under the name

**Interaction Philosophy**:
- Dark mode by default, optional light mode toggle
- Research cards expand on click to show full abstract
- Smooth parallax on hero section

**Animation**:
- Hero: name fades in letter by letter
- Cards: lift with box-shadow on hover
- Section transitions: horizontal slide-in from left

**Typography System**:
- Display: "Playfair Display" — classic, editorial weight
- Body: "Source Serif 4" — academic, readable
- Mono: "Fira Code" — for technical content
- Hierarchy: 64px hero / 36px h2 / 22px h3 / 16px body

## Idea 3: "Structured Intelligence" — Neo-Brutalist Academic (CHOSEN)

**Design Movement**: Neo-Brutalist minimalism meets academic precision — think a well-designed academic CV crossed with a modern tech portfolio

**Core Principles**:
1. Strong typographic structure — headings do the heavy lifting, no decorative noise
2. Asymmetric sidebar layout — persistent left nav, scrollable right content
3. Controlled color temperature — warm whites and cool slate with a single deep indigo-blue accent
4. Subtle tech texture — faint grid/dot pattern backgrounds, monospace accents

**Color Philosophy**:
- Background: #FAFAF8 (warm near-white, not clinical)
- Surface: #FFFFFF (cards, panels)
- Text: #1C2333 (deep slate-navy, professional)
- Accent: #2E4A8F (deep academic blue — finance, authority)
- Secondary: #5B8DD9 (lighter blue for links, tags)
- Muted: #6B7280 (gray for metadata, captions)
- Emotional intent: Trustworthy, intelligent, quietly ambitious

**Layout Paradigm**:
- Fixed left sidebar (240px) with name, photo, nav links, contact
- Right scrollable main content area
- Sections use large left-aligned headings with a blue left-border accent
- Research papers in a card grid with hover effects
- CV as a clean timeline

**Signature Elements**:
1. Subtle dot-grid background on hero/sidebar
2. Blue left-border on active nav items and section headings
3. Monospace font for years, tags, and technical metadata

**Interaction Philosophy**:
- Sidebar nav highlights current section on scroll (intersection observer)
- Research paper cards expand to show abstract on hover
- Smooth scroll with offset for fixed sidebar

**Animation**:
- Page load: sidebar slides in from left, content fades in from right (staggered)
- Hover: cards lift 2px with shadow deepening
- Section entry: fade-up with 40px translate, 0.4s ease

**Typography System**:
- Display: "Lora" (serif) — for name and section headings, academic gravitas
- Body: "Inter" — clean, universally readable (used sparingly, not as display)
- Mono: "JetBrains Mono" — for tags, years, technical labels
- Hierarchy: 36px name / 28px section h2 / 20px h3 / 15px body / 12px mono-label
